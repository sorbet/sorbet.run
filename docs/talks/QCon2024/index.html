What makes a good ratchet?

  • local
    • Sorbet: per-file
    • Dependencies: per-package
  • incremental
    • Sorbet: false to true to strict
    • Dependencies: false to layered to dag
  • actionable
    • Sorbet: ‚ÄúWhere do I need types in my current files?‚Äù
    • Dependencies: ‚ÄúWhat bad edges can I remove from my current package?‚Äù
